# Compare Examples

This folder contains equivalent web demo implementations in different languages/runtimes.

## Subfolders

- `cpp/`: C++ server + static frontend
- `js/`: Node.js server + static frontend
- `python/`: Python app + templates/static
- `rust/`: Rust web server + static frontend

## Purpose

- Compare implementation style and runtime behavior across stacks.
- Serve as reference for API shape and frontend behavior parity.

## Notes

- Keep existing local build/run artifacts in this folder (requested).
- When updating one implementation, align endpoint behavior in others if possible.
