Stability Check Deliverable (SeedLang)
======================================

This is not a Windows double-click app. Seed runs under Node from the repo.
The reliable check is the same command you already use to validate the language:

  From repo root (g:\Seed or your clone):
    npm test -- --quick

  That runs tests/deliverables/test-stability-deliverable.js among others,
  which compares app.seed interpreter output to golden.txt.

  Or only this check:
    node tests/deliverables/test-stability-deliverable.js

  Or:
    npm run verify:deliverable

Run app.seed by hand (interpreter; CLI prints twice):
    node dist/cli.js examples/deliverables/stability-check/app.seed

Optional CLC:
    node dist/cli.js examples/deliverables/stability-check/app.seed --compile-c

After editing app.seed, update golden.txt to match getOutput() lines, then re-run tests.
